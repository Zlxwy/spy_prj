# 道路508机器人赛事控制程序
## 项目概述
这是专门为青少年/大学生机器人工程挑战赛开发的巡线小车控制程序，基于xyrobot硬件平台实现，内置完整的巡线算法、机械动作控制和竞赛任务逻辑，可以快速组合出完整的比赛流程。

---
## 硬件依赖
1. 主控：xyrobot可编程机器人控制器
2. 传感器：5路灰度光电传感器（S1~S5，用于巡线检测）、双轮编码器
3. 执行器：双直流减速电机、2路舵机（分别控制夹爪升降、夹爪开合）

---
## 核心功能模块
### 1. 基础运动控制
| 功能 | 说明 |
| ---- | ---- |
| PID巡线算法 | 通过5路光电传感器的权重计算偏移量，自动调整左右轮速度实现精准沿黑线行驶 |
| 编码器控制 | 支持指定编码值的精准行驶，不走巡线也能控制移动距离 |
| 自定义转弯 | 支持指定左右轮转速，直到指定传感器检测到黑线停止，适配各种角度的转弯需求 |
| 单光电巡线 | 支持使用单个传感器沿黑线边缘行驶，适用于任务区的精准对位 |
| 刹车功能 | 支持反向制动，减少停车惯性提升定位精度 |

### 2. 机械夹爪控制
| 功能 | 说明 |
| ---- | ---- |
| 夹爪升降 | 控制7号舵机，支持匀速缓慢升降和直接跳转到指定角度两种模式 |
| 夹爪开合 | 控制10号舵机，支持匀速开合和直接跳转到指定角度两种模式，适配不同物料的抓取需求 |

### 3. 内置竞赛任务
程序已经内置了5类常见赛事任务的完整执行逻辑：
1. **物料回收**：将易拉罐物料投放到指定区域
2. **建设服务区**：抓取方块/杯子物料并运送到指定区域堆叠
3. **搭建桥梁**：抓取桥梁构件并安装到指定位置
4. **挖掘隧道**：通过双光电差分算法实现沿隧道两侧壁行驶
5. **建设加油站**：抓取加油站构件并组装到指定位置

### 4. 预设路口逻辑
内置A~L共12个场地路口的行驶逻辑，支持配置入口、出口、是否执行任务、任务参数、返回路线等，无需重复编写基础巡线代码，直接调用即可组合出完整比赛路线。

## 快速使用说明
### 1. 初始化配置
首先调用`设置()`函数配置传感器阈值和全局速度：
```python
设置(
    车号=1,
    S1高=1000, S1低=500,
    S2高=1000, S2低=500,
    S3高=1000, S3低=500,
    S4高=1000, S4低=500,
    S5高=1000, S5低=500,
    全局速度=60
)
```

### 2. 启动示例
以下是一个简单的比赛流程示例：
```python
# 启动前进到第一个路口
启动(启动距离=1100)

# 路口A，入口1，出口3，执行物料回收任务，放到1区
路口A(入口1=1, 出口1234=3, 任务0123=1, 区1=1, 完成返回路口=2)

# 路口B，入口2，出口3，不执行任务直接通过
路口B(入口1234=2, 出口1234=3, 任务0123=0)

# 路口C，执行挖掘隧道任务
路口C(入口1234=1, 出口1234=2, 任务0123=4)

# 终点返回充电区
路口L()
```

---
## 顶层API说明
| 函数名 | 功能说明 | 主要参数 |
| ---- | ---- | ---- |
| `设置()` | 初始化传感器阈值和全局参数 | 各传感器高低阈值、全局速度 |
| `启动()` | 比赛启动后初始前进 | 启动前进的编码距离 |
| `路口A() ~ 路口L()` | 对应场地各个路口的行驶逻辑 | 入口编号、出口编号、是否执行任务、任务参数、返回路口 |
| `_任务()` | 执行指定的竞赛任务 | 任务编号1~5，对应5个内置任务，各区参数 |

---
## 参数说明
### 区参数说明
所有任务的`区1`/`区2`/`区3`参数统一约定：
- < 2.5：靠左区域（1/2区）
- = 2.5：中间区域（2/3区之间）
- > 2.5：靠右区域（3/4区）
- 具体值1~4对应场地的1到4号区域

### 传感器编号说明
| 编号 | 传感器位置 |
| ---- | ---- |
| 1 | 最左侧传感器 |
| 2 | 左中传感器 |
| 3 | 中间传感器 |
| 4 | 右中传感器 |
| 5 | 最右侧传感器 |00

## 调试建议
1. 首次使用先单独校准每个传感器的黑白阈值，保证巡线稳定性
2. 可以将`isdebug`参数设为1，打开调试输出查看传感器值和PID计算结果
3. 低速调试通过后再逐步提升全局速度，避免速度过快冲出赛道
4. 每个任务单独测试通过后再组合到完整流程中
